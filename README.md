# Kawakumo

A browser extension that adds a streaming player to [The Movie Database (TMDb)](https://www.themoviedb.org) pages. When browsing movies or TV shows on TMDb, Kawakumo injects a play button that opens an embedded video player directly on the page — supporting both movies and individual TV episodes.

## Installation

Download the latest release for your browser from [GitHub Releases](https://github.com/adamgoodapp/kawakumo/releases).

### Chrome / Chromium-based browsers (Edge, Brave, Arc, etc.)

1. Download `kawakumo-chrome.zip` from the latest release
2. Unzip the file
3. Open `chrome://extensions` in your browser
4. Enable **Developer mode** (toggle in the top-right corner)
5. Click **Load unpacked**
6. Select the unzipped folder
7. The extension icon should appear in your toolbar

### Firefox

1. Download `kawakumo-firefox.zip` from the latest release
2. Open `about:addons` in Firefox
3. Click the gear icon and select **Install Add-on From File...**
4. Select the downloaded `.zip` file
5. The extension icon should appear in your toolbar

### Safari (macOS)

1. Download `kawakumo-safari.zip` from the latest release
2. Unzip the file
3. Move **Kawakumo.app** to your **Applications** folder and open it
4. In Safari, go to **Safari > Settings > Extensions** and enable Kawakumo
